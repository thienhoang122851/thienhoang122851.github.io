Cài đặt Bootloader

    OpenCore Bootloader EFI support CPUs Intel and CPUs AMD for Desktop or Laptop
  • Để cài OpenCore Bootloder lên USB hoặc ổ cứng, bạn chỉ việc giải nén EFI mình đã làm sẳn ở trên vào EFI của USB hoặc ổ cứng là xong
  • Nếu bạn cài OpenCore lên ổ cứng, bạn hãy reset BIOS để thêm OpenCore vào menu khởi động tránh những vấn đề sau này

Tinh chỉnh Config

  • Nếu Desktop dùng Intel CPU thì bạn hãy tạm thời bỏ qua bước này (trừ CPU Pentium và Celeron thì bạn cần làm thêm bước cuối mình đã hướng dẫn trong video ở dưới), còn AMD CPU thì bạn hãy xoá config.plist gốc ở /EFI/OC/ rồi rename config2.plist thành config.plist
  • Với Laptop thì bạn hãy vào /EFI/OC/config_laptop/, tìm đến config chứa tên iGPU của máy bạn, rename nó thành config.plist sau đó thay thế config gốc ở /EFI/OC/
